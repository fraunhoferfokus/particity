Release Notes

0.9.2 (20.08.2015)

  • Migration internal SVN -> public GitHub
  • Rename: particity
  • Use English as primary language throughout the documentation
  • Implement demonstration-mode with restricted/limited functionality

0.9.1 (03.06.2015)

  • Design/Layout der Suche überarbeitet
  • Diverse Korrekturen

0.9.0 (01.06.2015)

  • Tabellensortierung invertiert, aktuelle Datensätze zuerst
  • Verweis zu Passwort-Erinnerung und Registrierung neben dem Login
  • Zweites Kontaktfeld für Angebote, optionale Delegation des Kontakts an Agentur/Verwaltung
  • Bessere Handhabung von großen Bilddateien beim Logo-Upload von Organisationen
  • Konfiguration/Einstellungen i.d. Verwaltung besser strukturiert
  • Twitter/Facebook-Anbindung f.d. Verwaltung zum manuellen Absetzen (per Klick) von Posts bei neuen Angeboten
  • Suchresultate werden dynamisch nachgeladen, der Suchbutton entfällt bei aktiviertem Javascript

0.8.4 (20.05.2015)

  • Land als Liste beim Registrierung von Organisationen und in Angeboten
  • Arbeitszeit in Angeboten als Liste
  • Kategorien für Leistungen der Organisationen in Angeboten
  • Mehrere Benutzer pro Organisation
  • Tabellen im Backend für Organisation/Verwaltung sind nach Spalten sortierbar
  • Login immer sichtbar (aktuell: Footer)

0.8.3 (08.05.2015)

  • Suchdetails enthalten Kurzverweise auf Vorgänger/Nachfolger in den Suchresultaten
  • Suchdetails enthalten Verweis auf den Permalink; externe URLs öffnen in neuem Tab
  • Verwaltung/Organisation: Direktlinks auf Titel von Einträgen verweisen auf Detailseite
  • Verwaltung/Organisation: Abfrage über Bestätigungsdialog beim Löschen von Organisationen/Angeboten
  • Organisationen können nach der Registrierung ein Logo hochladen (optional)
  • Suchdetails enthalten einen Button zum Auflisten aller Angebote einer Organisation
  • Neue Datenlisten im Backend (Länder, Arbeitsstunden f. Angebote, Kategorien für Angebotsleistungen)

0.8.2 (10.04.2015)

  • Angebote werden im Backend erst nach Status, dann nach Änderungsdatum gelistet
  • Verwaltungs-Mail-Adresse & Name sind jetzt in der Verwaltungskonfiguration einstellbar
  • Nutzer/Address-Export für die Verwaltung möglich
  • Organisationen erhalten eine Erinnerung, falls in der letzten Woche keine Angebote eingestellt wurden
  • Organisationen erhalten eine Nachricht, wenn Angebote in der letzten Stunde ausgelaufen sind
  • Organisationen erhalten eine Nachricht nach Einstellen eines neuen Angebots (vor der Moderation)
  • Verwaltungs-Tabs zeigen über Info-Badges, wie viele offene Aufgaben anstehen
  • Das System überprüft die Datenbank alle 10 Minuten auf neue Angebote und stellt abonnierte Newsletter zu
  • Angebotdetails verwenden einen lesbaren Permalink und werden in Newsletter-Mails direkt verlinkt

0.8.1 (01.04.2015)

  • Korrektur: Fehlerhafte Darstellung beim Blättern durch Suchergebnisse
  • Geändert: Nummerierte Anzeige der Suchergebnisse als Marker-Grafik
  • Hinzugefügt: Social-Media Buttons in Suchdetails
  • Geändert: Angebote ohne korrekte Geo-Koordinaten werden beim Erstellen abgelehnt

0.8.0 (30.03.2015)

  • öffentlicher Zugang
