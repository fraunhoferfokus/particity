Project Modules

This project has declared the following modules:

Name Description
lib_data Shared data library, carrying all data structures and basic service interfaces
hook_securefilter Hook providing filters to enhance liferay security
portlet_data Main portlet. carrying basic services and the administrative frontend UI
portlet_search Search portlets
portlet_mgmt Portlets for managing organisations
portlet_org Organisation portlets - registration, internal management, etc
portlet_newsletter Newsletter registration portlet
portlet_site Various site portlets, e.g. login, profile, etc.
portlet_init Portlet for first-time initialization providing a default setup of the platform
layout_2col Variations of a bootstrap layout with 2 columns of different width
layout_3col Variations of a bootstrap layout with 1,2,3 or 4 columns of equal width
theme_default Liferay theme based upon bootstrap 3.x, including JQuery, modernizr, respond and shariff
